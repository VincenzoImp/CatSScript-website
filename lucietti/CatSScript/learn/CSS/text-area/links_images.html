Links and images


HTML Links - Hyperlinks

HTML links are hyperlinks: you can click on a link and jump to another document.
A link does not have to be text: a link can be an image or any other HTML element.

The HTML <a> tag defines a hyperlink. It has the following syntax:
<a href="url">link text</a>

The most important attribute of the <a> element is the href attribute, which indicates the link's destination

The link text is the part that will be visible to the reader; clicking on the link text, will send the reader to the specified URL address.

For example if you want to create a link to CatSScript you should use the following tag:
<a href="../index.html>Visit CatSScript!</a>

By default, links will appear as follows in all browsers:

  • An unvisited link is underlined and blue
  • A visited link is underlined and purple
  • An active link is underlined and red

Of course this default visualization can be customized through CSS.



Absolute URLs and Relative URLs

You can use absolute URL or relative URL in the href attribute.
You must use an absolute URL, a full web address, if you want to link a Web page.
You must use a relative URL, an address without the "https://www", if you want to link a page within the same website.
Note that you can link also a different section of the same page just using <a href="#idSection>



HTML Images

The <img> tag is used to embed an image in a web page.
Images are not really inserted into a web page but are linked to web pages: the <img> tag creates a holding space for the referenced image.

The <img> tag is empty, it contains attributesd only and doea not have a closing tag.

The <img> tag has two required attributes:

  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image

The syntax of the <img> tag is:
<img src="url" alt="alternatetext">


The src Attribute

The required src attribute specifies the path (URL) to the image

The alt Attribute

The required alt attribute provides an alternate text for an image, if the user for some reason cannot view it.
The value of the alt attribute should describe the image.
Note that the alt attribute is also used by screen readers, programs that reads the HTML code, and allows the user to "listen" to the content. Screen reader are useful for people who are visually impaired or learning disabled.
