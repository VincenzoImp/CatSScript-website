Introduction and first steps


What is CSS?

CSS is the language used to style an HTML document.
CSS describes how the HTML elements should be displayed.

CSS stands for Cascading Style Sheets.
The same HTML page can use multiple CSS style sheets.

CSS was created to describe the content of a web page, to format a web page.

Style sheets are a collection of CSS rules.

What is a CSS Rule?

A CSS rule consists of a selector and a declaration block.
The selector points to the HTML element you want to style, while the declaration block contains one or more declarations separated by semicolons.
Each declaration includes a CSS property name and a value, separated by a colon (note that for all those values that need an unit you must not add a space between the value and the unit).
Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.
For example, if you want to write <p> elements center-aligned with a red text color, you should write the following CSS rule: 

p {
color: red;
text-align: center;
}

In this example we have that:

  • p is a CSS selector and it points to the <p> elements in the HTML document;
  • color is a property, and red is the property value;
  • text-align is a property, and center is the property value.


How to add CSS

There are three ways to insert a style sheet in a HTML document:

  • External CSS
    To use an external style sheet you must reference the external style sheet file inside the <link> element inside the head section.
    An external style sheet must be saved with a .css extension and should not contain any HTML tags.
  • Internal CSS
    An internal style sheet may be used if one single HTML page has a unique style.
    The internal style sheet is defined inside the <style> element, inside the head section.
  • Inline CSS
    An inline style may be used to apply a unique style for a single element.
    To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.
Note that id some properties have been defined for the same selector in different style sheets, the value from the last read style sheet will be used.
Also note that all the styles in a page follow this hierarchy (number one has the highest priority):
  1. Inline style (inside an HTML element)
  2. External and internal style sheets (in the head section)
  3. Browser default
