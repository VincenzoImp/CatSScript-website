Headings and paragraphs


HTML Headings

Headings are titles or subtitles that you want to display on a webpage.

HTML headings are defined with the <h1> to the <h6> tags. <h1> defines the most important heading, <h6> defines the least important heading.
Note that browsers automatically add some white space before and after a heading.

Here's an example of what each heading shows:

  • <h1>Heading 1</h1> will produce

    Heading 1

  • <h2>Heading 2</h2> will produce

    Heading 2

  • <h3>Heading 3</h3> will produce

    Heading 3

  • <h4>Heading 4</h4> will produce

    Heading 4

  • <h5>Heading 5</h5> will produce
    Heading 5
  • <h6>Heading 6</h6> will produce
    Heading 6


Headings are really important because search engines use them to index the structure of web pages (they are also important because they help the user to understand the structure of a page).



HTML Paragraphs

The <p> element defines a paragraph.
A paragraph always starts with a new line and browsers automatically add some white space before and after it.

The browser will automatically remove any extra spaces and lines when the page is displayed, so the following code: 

  1. <p>
  2. This paragraph
  3. contains         a lot of spaces
  4. in the source         code,
  5. but the        browser
  6. ignores it.
  7. </p>

will actually produce:
This paragraph contains a lot of spaces in the source code but the browser ignores it.

To display breaks to separate content you can use either <hr> or <br> tags.



HTML Breaks

The <hr> tag

The <hr> tag defines a thematic break in an HTML page and is displayed as an horizontal line. For example the following code: 

  1. <h1>This is heading 1</h1>
  2. <p>This is some text.</p>
  3. <hr>
  4. <h2>This is heading 2</h2>
  5. <p>This is some other text.</p>
  6. <hr>

will produce:

This is heading 1

This is some text.

This is heading 2

This is some other text.

Note that the <hr> tag is an empty tag, which means that it has no end tag.


The <br> tag

The HTML <br> elements defines a line break.
Use <br> if you want a line break without starting a new paragraph. For example the following code: 

  1. <p>This is<br>a paragraph<br>with line breaks.</p>

will produce:

This is
a paragraph
with line breaks.

Note that the <br> tag is an empty tag, which means that it has no end tag.


The <pre> tag

The <pre> elements defines preformatted text: this means that the text inside a <pre> tag is displayed in a fixed-width font and it preserves both spaces and line breaks. For example the following code: 

  1. <pre>
  2.   My Bonnie lies over the ocean.
  3.  
  4.   My Bonnie lies over the sea.
  5.  
  6.   My Bonnie lies over the ocean.
  7.  
  8.   Oh, bring back my Bonnie to me.
  9. </pre>

will produce: 

        My Bonnie lies over the ocean.
        
        My Bonnie lies over the sea.
        
        My Bonnie lies over the ocean.
        
        Oh, bring back my Bonnie to me.
