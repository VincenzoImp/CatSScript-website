Tables, lists and containers


HTML Tables

HTML tables aallow web developers to arrange data into rows and columns.
To define an HTML table you can use the <table> tag: each table row is defined with a <tr> tag; each table header is defined with a <th> tag; each table cell is defined with a <td> tag.

By default, the text in <th> elements are bold and centered and the text in <td> elements are regular and left-aligned.

Here's an example of a simple table:

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

Note that <td> elements are data containers for the table so they can contain all sorts of HTML elements like text, image, lists, other tables, etc.

By default tables are borderless: to add a border you can use CSS.

You can create tables in which a cell span more than one column or row with the <colspan> and <rowspan> attributes. Here's an example (and it's HTML code) of a table with a cell that has two columns: 

  1. <table style="width:100%">
  2.   <tr>
  3.     <th>Name</th>
  4.     <th colspan="2">Telephone</th>
  5.   </tr>
  6.   <tr>
  7.     <td>Bill Gates</td>
  8.     <td>55577854</td>
  9.     <td>55577855</td>
  10.   </tr>
  11. </table>

Name Telephone
Bill Gates 55577854 55577855

Here's an example (and it's HTML code) of a table with a cell that has two rows: 

  1. <table style="width:100%">
  2.   <tr>
  3.     <th>Name:</th>
  4.     <td>Bill Gates</td>
  5.   </tr>
  6.   <tr>
  7.     <th rowspan="2">Telephone:</th>
  8.     <td>55577854</td>
  9.   </tr>
  10.   <tr>
  11.     <td>55577855</td>
  12.   </tr>
  13. </table>

Name: Bill Gates
Telephone: 55577854
55577855

Note that you can also add a caption with the <caption> tag; this tag must be inserted immediatly after the <table> tag. Here's an example of a table with a caption:

Monthly savings
Month Savings
January $100
February $50



HTML Lists

There are two types of lists in HTML: unordered lists and ordered lists.
Of course HTML lists can be nested: in fact a list item can contain a new list or other HTML elements like images, links , etc.

Unordered HTML List

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.
The list items will be marked with bullets by default.
You can change your marker through CSS.
Here's an example of an unordered list:

  • Coffee
  • Tea
  • Milk

Ordered HTML List

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.
The list items will be marked with numbers by default.
You can change your markers through the <type> attribute of the <ol> tag; you can have the following markers:

Type Description
type="1" The list items will be numbered with numbers (default)
type="A" The list items will be numbered with uppercase letters
type="a" The list items will be numbered with lowercase letters
type="I" The list items will be numbered with uppercase roman numbers
type="i" The list items will be numbered with lowercase roman numbers

Here's an example of an oredered list:

  1. Coffee
  2. Tea
  3. Milk

Note that by default, an ordered list will start counting from 1. If you want to start counting from a specified number, you can use the <start> attribute of the <ol> tag


HTML Containers

There are two types of HTML containers: the <div> element and the <span> element.

The <div> Element

The <div> element is used as a container for other HTML elements.
The <div> elements has no required attributes.
When used together with CSS (or the <style> attribute), the <div> element can be used to style blocks of content.
Here's an example (with it's HTML code) of what said: 

  1. <div style="background-color:black;color:white;padding:20px;">
  2.   <h2>London</h2>
  3.   <p>London is the capital city of England. It is the most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitants.</p>
  4. </div>

that produces:

London

London is the capital city of England. It is the most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitants.


The <span> Element

The <span> element is an inline container used to markup a part of text or a part of a document.
The <span> element has no required attributes.
When used togheter with CSS (or the <style> attribute), the <span> element can be used to style parts of the text.
Here's an example (and it's HTML code) of what said: 

  1. <p>My mother has <span style="color:blue;font-weight:bold">blue</span> eyes and my father has <span style="color:darkolivegreen;font-weight:bold">dark green</span> eyes.</p>

that produces:

My mother has blue eyes and my father has dark green eyes.
