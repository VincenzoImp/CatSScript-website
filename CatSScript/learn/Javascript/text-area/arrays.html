JavaScript Arrays

JavaScript arrays are used to store multiple values in a single variable.


What is an Array?

An array is a special variable which can hold more than one value at a time.
Arrays are used to loop through a list of items because an array can hold many values under a single name and you can access the values by referring to an index number.


Creating an Array

Using an array literal is the easiest way to create a JavaScript Array.
The syntax is the following:
var array_name = [item1, item2, ...];
Spaces and line breaks are not important so you can span a declaration on multiple lines to make your code more readable.

You can also use the keyword new to create an Array, for example:
var person = new Array("Mario", "Luigi", "Anna")


Accessing the Elements of an Array

You can access an array element by referring to the index number.
The syntax is the following:
var varName = arrayName[i]
Note that array indexes start with 0

To change an array element you can use assignement; the syntax is the following:
arrayName[i] = newValue


Array Properties and Methods

There are some bulit-in methods that are very useful to manipulate arrays.
The most important are:

  • arrayName.length()
    This method returns the value of the length property which tells you how many elements are into an array
  • arrayName.sort()
    This method sorts the element of the array
