JavaScript Objects

Objects are variables that can contain many values.
The values are written as name:values pairs (where eache pair is separated by a colon from the others).
For example: 

  1. var person = {name:"Mario", surname:"Rossi", age:"32"};

The code above assigns many values (Mario, Rossi, 32) to a variable named person.

JavaScript objects are containers for named values called properties or methods.


Object Definition

You define and create a JavaScript object with an object literal like 

  1. var person = {name:"Mario", surname:"Rossi", age:"32"};

Space and line breaks are not important, so you can span multiple lines to make your code more readable.
So this 

  1. var person = {
  2.     name:"Mario", 
  3.     surname:"Rossi", 
  4.     age:"32"
  5. };

has the same meaning of the same code written on one line.


Object Properties and Accessing Object Properties

The name:values pairs in JavaScript objects are called properties.

You can access object properties in two ways:
objectName.propertyName
or
objectName["propertyName"]


Object Methods and Accessing to Object Methods

Objects can also have methods: methods are actions that can be performed on objects.
Methods are stored in properties as function definitions.
For example: 

  1. var person = {
  2.   name: "Mario",
  3.   surname : "Rossi",
  4.   fullName : function() {
  5.     return this.name + " " + this.surname;
  6.   }
  7. };

Note that in the code above, the keyword this is used.
In a function definition, this refers to the "owner" of the function.

You can access an object method with the following syntax:
objectName.methodName()

Note that if you access a method without the () parentheses, it will return the function definition
