Introduction and first steps


What is JavaScript?

JavaScript is the programming language of the web.
It is used to program the behavior of web pages


JavaScript Functions and Events

A JavaScript script is made of function.
A function is a block of JavaScript code, that can be executed when "called" for: most of the JavaScript Functions are event handler so they are called when an event (like the click of a user on a button) occurs.


How to add JavaScript scripts

In an HTML document, JavaScript code is inserted between <script> and </script> tags.
If you want to link an external script you can use the following syntax: <script src="script_file_url"></script>

Scripts can be placed in the <body>, or in the <head> section of an HTML page, or in both.
Usually placing scripts at the bottom of the <body> element improves the display speed, because script interpretation slows down the display; placing them at the end of the body section also helps them to work correctly because their interpretation is done after the whole web page has been charged.


JavaScript Display Possibilities

JavaScript can "display" data in different ways:

  • Writing into an HTML element, using innerHTML
    To access an HTML element, JavaScript can use the document.getElementById(id) method.
    The id attribute defines the HTML element. The innerHTML property defines the HTML content.
  • Writing into the HTML output using document.write()
  • Writing into an alert box, using window.alert()
    This method is used to create an alert box to display data.
    Note that you can skip the window keyword.


JavaScript Statements

JavaScript statements are composed of Values, Operators, Expressions, Keywords and Comments.

The statements are executed, one by one, in the same order as they are written.
Semicolons separate JavaScript statements, this means that you must add a semicolon at the end of each executable statement.
When separated by semicolons, multiple statements on one line are allowed.

JavaScript ignores multiple spaces.
