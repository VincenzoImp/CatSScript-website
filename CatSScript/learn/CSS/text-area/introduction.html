Introduction and first steps


What is HTML?

HTML stands for Hyper Text Markup Language, it's the standard markup language for creating Web pages and describes the structure of a Web page.
HTML consists of a series of elements that tell the browser how to display the content.

What is a HTML element?

An HTML element is defined by a start tag, some content and an end tag:
<tagname>Content goes here...</tagname>
Some HTML element have no content: these elements are called empty elements and they do not have the end tag!

HTML elements can be nested: this means that elements can cointain other elements (in fact, all HTML docuemnts consists of nested elements).

All HTML elements can have attributes that provide additional information about HTML elements; attributes are always specified in the start tag and usually come in name/value pairs like: name="value".

Every HTML element has a default display value, depending on what type of element it is; there are two display values: block and inline.
Block-level elements always start on a new line and always take up the full width available.
A block-level element has a top and a bottom margin and can contain an inline element.
Inline elements do not start on a new line and only take take up as much width as necessary.
An inline element does not have any margin and cannot contain a block-level element



HTML page structure

Below is a visualization of a web page:

<html>
<head>
<title>Page title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
</body>
</html>

The content inside the <body> section will be displayed in a browser, the content inside the <title> element will be shown in the browser's title bar or in the page's tab.



HTML Basics

HTML Documents

Here's an example of the structure of a web page: 

  1. <!DOCTYPE html>
  2. <html>
  3.     <head>
  4.         ...
  5.     </head>
  6.     <body>
  7.         ...
  8.     </body>
  9. </html>

All HTML documents must start with a document type declaration: <!DOCTYPE html>
The document type declaration represents the docuemtn type and helps the browser to display it correctly; it must appear only once at the top of the page, before any HTML tags.
The <!DOCTYPE html> declaration is not case sensitive.

The document itself begins with <html> and ends with </html>

The part of the document between <head> and </head> is invisible and contains information and metadata.

As said before, the visible part of the document is between <body> and </body>.
