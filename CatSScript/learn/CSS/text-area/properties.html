CSS Properties

Here are listed some of the most important CSS properties:

  • color
    This property is used to style the color of a text
  • background-color
    This property specifies the background color of an element
  • background-image
    This property specifies an image to use as the background of an element.
    By default, the image is repeated so it covers the enitre element.
  • border: border-width border-style border-color
    This property is used to set a border with a thickness equal to border-width, with the style choosen by border-style whose color is set by border-color.
    The thickness of the line can be expressed in pixel.
    The border-style property specifies what kind of border to display; these are the style allowed:
    • dotted - Defines a dotted border
      A dotted border

    • dashed - Defines a dashed border
      A dashed border

    • solid - Defines a solid border
      A solid border

    • double - Defines a double border
      A double border

    • groove - Defines a 3D grooved border. The effect depends on the border-color value
      A groove border

    • ridge - Defines a 3D ridged border. The effect depends on the border-color value
      A ridge border

    • inset - Defines a 3D inset border. The effect depends on the border-color value
      An inset border

    • outset - Defines a 3D outset border. The effect depends on the border-color value
      An outset border

    • none - Defines no border
      No border

    • hidden - Defines a hidden border
      A hidden border

    Note that borders can also be mixed
    A mixed border
    Also note that is possible to specify a style for each border (top, right, bottom and left)
  • border-radius
    This property is used to add rounded borders to an element
  • margin
    This property is used to create space around elements, outside of any defined borders.
    There are proeprties for setting the margin of each side of an element (margin-top, margin-right, margin-bottom, margin-left).
    Note that all margin properties can have the following values:
    • auto - the browser calculates the margin
    • length - specifies a margin in px, pt, cm, etc.
    • % - specifies a margin in % of the width of the containing element
    • inherit - specifies that the margin should be inherited from the parent element
    Also note that negative values are allowed
  • padding
    This property is used to generate space around an element's content, inside of any defined borders.
    Ther are properties for setting the padding of eache side of an element (padding-top, padding-right, padding-bottom, padding-left).
    Note that all padding properties can have the following values:
    • auto - the browser calculates the padding
    • length - specifies a padding in px, pt, cm, etc.
    • % - specifies a padding in % of the width of the containing element
    • inherit - specifies that the padding should be inherited from the parent element
    Also note that negative values are allowed
  • height and width This property is used to set the height or the width of an element.
    The height and the width properties may have the following values:
    • auto - This is default. The browser calculates the height/width
    • length - Defines the height/width in px, cm, etc.
    • % - Defines the height/width in percent of the containing block
    • initial - Sets the height/width to its default value
    • inherit - The height/width will be inherited from its parent value
  • text-align This property is used to set the horizontal alignment of a text.
    A text can be left or right aligned, centered or justified.
  • text-decoration This property is used to set or remove decorations form text.
    The value text-decoration: none is often used to remove underlines from links

Note that colors in CSS can be specified by a valid color name, a HEX value or an RGB value.
