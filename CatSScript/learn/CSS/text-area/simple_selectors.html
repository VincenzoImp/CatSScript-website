CSS Simple Selectors

The CSS element Selector

The element selector selects HTML elements based on the element name.
For example: 

p {
  color: red;
  text-align: center;
}

Here all <p> elements are styled to be center-aligned and with a red text.


The CSS id Selector

The id selector selects HTML elements based on the id attribute of an HTML element.
The id of an element is unique within a page, so the id selector is used to select one unique element.
To select an element with a specific id, write a hash character (#), followed by the id of the element.
For example: 

#para1 {
  text-align: center;
  color: red;
}

The CSS rule above will be applied to the HTML element with id="para1".
Note that an id name cannot start with a number.


The CSS class Selector

The class selector selects HTML elements with a specific class attribute.
To select elements with a specific class, write period character (.), followed by the class name.
For example: 

.center {
  text-align: center;
  color: red;
}

The CSS rule above will be applied to all HTML elements with class="center".
The rule will format all the HTML elements selected with red text and center-aligned.

Note that you can also specify that only specific HTML elements should be affected by a class.
For example: 

p.center {
  text-align: center;
  color: red;
}

The CSS rule above will be applied only to <p> elements with class="center".

Note that HTML elements can also refer to more than one class and so they can be influenced by more than one CSS rule.


The CSS Universal Selector

The universal selector (*) selects all HTML elements on the page.
For example: 

* {
  text-align: center;
  color: blue;
}

The CSS rule above will affect every HTML element on the page


The CSS Grouping Selector

The grouping selector selects all HTML elements with the same style definitions.
It is fundamentally used to minimize the code so you don't have to write the same CSS rules more than one time.
For example: 

h1, h2, p {
  text-align: center;
  color: red;
}

The CSS rule above is the same and will affect all <h1>, <h2> and <p> elements.
