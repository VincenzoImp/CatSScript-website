CSS Pseudo-class Selectors

What are Pseudo-classes?

A pseudo-class is used to define a special state of an element.
For example, it can be used to:

  • Style an element when a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus
The syntax of pseudo-classes is the following: 

selector:pseudo-class {
property: value;
}

Anchor Pseudo-classes

Links can be displayed in different ways, so you can use the following pseudo-classes in your stylesheet to display them in a different way for every different situation:

  • link: with this pseudo-class you will decide how an unvisited link is displayed
  • visited: with this pseudo-class you will decide how a visited link is displayed
  • hover: with this pseudo-class you will decide how a link is displayed when a user mouses over it
  • active: with this pseudo-class you will decide how a selected link is displayed
Note that a:hover must come after a:link and a:visited in the CSS definition in order to be effective.
Also note that a:active must come after a:hover in the CSS definition in order to be effective.
In the end note that pseudo-class names are not case-sensitive.
